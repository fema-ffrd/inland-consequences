# Basic Install Guide

This guide walks you through downloading the repository and setting up your environment so you can run the prototype example notebooks included with the Consequences Solution.

!!! tip "Detailed Walkthrough"
    A step-by-step PDF guide is included in the repository at
    `examples/ConsequenceSolution_HowToDownloadLaunchPrototypeNotebook.pdf`.
    It covers environment setup, launching notebooks, and working with the sample data.

![Basic Install Guide Cover](images/InstallGuideCover.png)

For other install workflows — including installing the package directly from a GitHub Release — see [Install from GitHub Release](gh_installation.md).
